# AI Agent System

This folder contains the document set for operating the SoundCheck / Church Audio Guard project with seven AI agents.

## Core Files

- `AI_Agent_Operating_System.md`: Overall operating system
- `agent-cards/`: Role cards for the seven agents
- `tasks/initial_task_queue.md`: Initial seven-day task queue
- `tasks/decision_log.md`: Operating decision log
- `templates/`: Report and task card templates

## Seven Agents

1. Project Director
2. Product Lead
3. Customer / Market Agent
4. Product / Content Agent
5. Engineering Lead
6. Audio Engine Agent
7. App / QA Agent

## How to Use

1. Give `AI_Agent_Operating_System.md` to the manager agent as the reference document.
2. Use `agent-cards/00_project_director.md` as the manager agent's starting prompt.
3. Give each working agent its matching file from `agent-cards/`.
4. Start automated work from Day 1 using `tasks/initial_task_queue.md`.
5. Send only the manager agent's daily report to the user.
